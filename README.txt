В папке «витрина данных» содержатся две витрины данных по ветки с красным шламом(«GMC_red_ch.csv») и алюминатным раствором(«GMC_al.csv»)
В папке «scripts» содержатся скрипты по обработки исторических данных, которые собирают финальную витрину, обучают модели и обеспечивают работоспособность сервиса на портале:
«preprocessing.ipynb» обрабатывает входные данные и собирает витрину
«GMC_model_v7.ipynb» обучает модели для первого и второго типа сценария
«GMC_model_v8.ipynb» обучает модель с дозировкой оборотного раствора
«GMC_model_v9.ipynb» обучает модели для третьего сценария
«GMC_model_v7_jel.ipynb» обучает модели для железных песков
«GMC_for_docker.ipynb» обеспечивает работоспособность расчетов на портале
В папке «model_boxit» содержится модель для дозировки оборотного раствора 
В папке «model» содержатся модели для расчета первого и второго типа сценария
В папке «model_5» содержатся модели для железных песков
В папке «model_2018» содержатся модели для третьего типа сценария
В папке «data» содержатся исторические данные по всем ГМЦ, ЦС и хим. анализам
Файл «Объектная модель ПАЗ.xlsx» содержит вспомогательную информацию
